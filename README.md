# Sakura Ramen — Responsive Landing Page

Ein modernes, responsives Landing-Page-Layout für ein fiktives Ramen-Restaurant mit klarer Struktur, ansprechender Typografie und einer mobiloptimierten Benutzeroberfläche.

## 📌 Projektübersicht

- **Titel:** Sakura Ramen
- **Technologien:** HTML5, CSS3
- **Ziel:** Präsentation einer Restaurant-Homepage mit Menü, Bestellablauf, Standort und Kontaktoptionen.
- **Design:** Fullscreen Hero-Sektion, klare Abschnittsaufteilung, responsive Navigation und eingebettete Karte.

## ✨ Features

- Voll responsive Design für Desktop, Tablet und Mobilgeräte
- Hero-Sektion mit großem Hintergrundbild und markanter Typografie
- Inhaltsabschnitte:
  - `HOW TO ORDER`
  - `OUR MENU`
  - `FIND US AT`
- Integrierte Kontakt-Schaltfläche mit `tel:`-Link
- Google Fonts: `Raleway` und `Economica`
- Externe Links zu Social Media und TripAdvisor in der Fußzeile
- Eingebettete Google Maps Karte für einen Restaurant-Standort in München

## 📁 Projektstruktur

- `index.html` — Hauptseite
- `style.css` — Layout, Typografie und responsive Stile
- `img/` — Bildressourcen für Hero, Menü, Icons und Footer

## 🚀 So lokal ausführen

1. Repository klonen oder Projektordner öffnen
2. `index.html` im Browser öffnen

> Es ist keine zusätzliche Installation notwendig, da das Projekt nur aus statischen HTML- und CSS-Dateien besteht.

## 💡 Hinweise

- Für eine Live-Vorschau reicht das lokale Öffnen der Datei im Browser.
- Optional kann ein einfacher HTTP-Server verwendet werden, z. B. `python3 -m http.server` im Projektordner.
